# Earth Time-Lapse Builder 🌍

A web application that allows users to create stunning time-lapse videos using NASA's EPIC (Earth Polychromatic Imaging Camera) satellite imagery. This tool provides an intuitive interface to select date ranges and generate beautiful visualizations of Earth from space.

## Features ✨

- Interactive date range selection for time-lapse creation
- Real-time preview of selected satellite images
- Beautiful, modern UI with a space-themed design
- Quick selection presets for common time ranges
- Progress tracking for image processing
- Responsive design that works on all devices

## Technologies Used 🛠

- HTML5
- CSS3 (with modern features like CSS Grid and Flexbox)
- JavaScript
- NASA EPIC API

## Getting Started 🚀

1. Clone the repository:
```bash
git clone https://github.com/yourusername/earth-time-lapse-builder.git
```

2. Open the `Earth Time-Lapse Builder.html` file in your web browser.

3. Select your desired date range or use the quick selection presets.

4. Click the "Generate Time-Lapse" button to create your video.

## Usage 📝

1. **Date Selection**: Choose your desired start and end dates for the time-lapse.
2. **Quick Selection**: Use the preset buttons for common time ranges (Last Week, Last Month, etc.).
3. **Image Preview**: View the selected satellite images in the grid below.
4. **Video Generation**: Click the generate button to create your time-lapse video.
